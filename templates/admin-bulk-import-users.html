Required Columns:
  • name - Full name of the user
  • email - Email address (must be @sjec.ac.in)
  • password - Initial password for the user
  • role - User role (see valid roles below)
  • department - Department name (required for faculty/hod)
Valid Roles:
  • faculty - Faculty member (requires department)
  • hod - Head of Department (requires department)
  • dean - Dean (no department)
  • dean_secretary - Dean Secretary (no department)
  • principal - Principal (no department)
  • vice_principal - Vice Principal (no department)
  • admin - Administrator (no department)
  • director - Director (no department)
Important:
  • Email addresses must be unique
  • Department must exist in the system
  • Faculty and HOD roles require a department
  • Admin roles cannot have a department
