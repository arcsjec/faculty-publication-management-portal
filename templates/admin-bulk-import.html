Instructions
Excel File Format:

Your Excel file must have the following columns:

  • faculty_email (required) - Email of the faculty member
  • title (required) - Publication title
  • authors_names (required) - Comma-separated author names
  • year (required) - Publication year
  • abstract - Publication abstract
  • publisher_name - Name of publisher
  • journal_conference_name - Journal or conference name
  • volume - Volume number
  • issue - Issue number
  • pages - Page numbers
  • indexing_status - e.g., Scopus, Web of Science
  • quartile - e.g., Q1, Q2, Q3, Q4
  • impact_factor - Impact factor (number)
  • doi - Digital Object Identifier
  • month - Month (1-12)
  • citation_count - Number of citations
  • bibtex_entry - BibTeX entry
Important Notes:
  • All imported publications will be automatically confirmed
  • The faculty_email must match an existing user in the system
  • Column names must match exactly (case-sensitive)
  • First row should contain column headers
  • Empty cells are allowed for optional fields
Warning: Large imports may take several minutes. Do not close this page until the import is complete.
