Best Practices
Database Optimization:
  • Run optimization monthly or when database performance degrades
  • Optimization creates indexes on frequently queried fields for faster searches
  • Updates database statistics for better query planning
Backup Strategy:
  • Daily backups: Use the scheduled backup feature (Windows Task Scheduler)
  • Before major changes: Always create a manual backup
  • Retention: Keep at least 30 days of backups
  • Storage: Store backups in a separate location for safety
Recovery:
  • To restore from backup, download the backup file
  • Stop the application
  • Replace instance/faculty_publications.db with the backup file
  • Restart the application
Setup Scheduled Backup
Automated Backup Script Ready! Follow the steps below to enable daily backups.
Windows Task Scheduler Setup:
  1. Open Task Scheduler (search in Windows Start menu)
  2. Click "Create Basic Task" in the right panel
  3. Name: SJEC Publication Portal Daily Backup
  4. Trigger: Daily at a time when portal usage is low (e.g., 2:00 AM)
  5. Action: Start a program
  6. Program/script: Browse and select run_backup.bat from the portal folder
  7. Start in: Enter the full path to the portal folder (e.g., C:\path\to\portal)
  8. Click Finish
Manual Backup (Alternative):

Run run_backup.bat manually anytime to create a backup and clean old backups.
